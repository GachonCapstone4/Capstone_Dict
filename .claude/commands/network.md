## Network 모듈

본 spec은 물리 Network 의 점검을 담당한다. 
본 진단 자동화는 총 3단계로 구성된다.
1. vmbr 내부 ARP 테이블 점검
2. pfSense(게이트웨이) ICMP 도달성 점검
3.  외부 IP(8.8.8.8) ICMP 점검

즉 L2 -> L3 -> 외부 통신 순서로 점검한다.

### 1. 실행 시작시
"
================================
($해당노드 ip)의 물리 네트워크 점검을 시작합니다....
================================
"
출력, 해당 시점에서 해당 출력 json 화 후 RabbitMQ 전송.

### 2. 두번째 단계
vmbr 내부 ARP 테이블 점검한다.
Node[0] → Node[1] → Node[2] → Node[0] (192.168.2.10 -> 192.168.2.20 -> 192.168.2.30 -> 192.168.2.10 )
같은 vmbr 내에서 Ring 구조 체크
====================================================================
($해당노드 ip)에서 ($해당노드 ip)에서 Birde 내부 ARP 테이블을 점검합니다...
====================================================================
출력, 해당 시점에서 해당 출력 json 화 후 RabbitMQ 전송.

실제 로직수행

=================================================================
"($해당노드 ip)에서 ($해당노드 ip)에서 Birde 내부 ARP 점검 결과 출력"
===============================================================

출력, 해당 시점에서 해당 출력 json 화 후 RabbitMQ 전송.
